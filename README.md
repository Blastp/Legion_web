# Legionarios Football Americano - Website

Sitio web oficial del equipo Legionarios Football Americano de Argentina.

## Características

- Información del equipo y participación en FABA
- Calendario de eventos y torneos
- Tienda online con carrito de compras
- Contacto directo por WhatsApp
- Diseño responsive

## Estructura del Proyecto

```
Legion_web/
├── index.html          # Página principal
├── shop.js            # Funcionalidad de la tienda
├── cart.js            # Sistema de carrito (backup)
├── styles.css         # Estilos CSS
├── imagesLGN/         # Imágenes del equipo
└── README.md          # Este archivo
```

## Despliegue

Este sitio está listo para ser desplegado en cualquier servicio de hosting estático como:

- Netlify
- Vercel
- GitHub Pages
- Firebase Hosting
